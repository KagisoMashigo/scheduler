## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node express_server.js or npm start if using nodemon` command.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- method-override

## Dev Dependencies

- mocha
- chai
- nodemon

Have fun shortening them URLs!
